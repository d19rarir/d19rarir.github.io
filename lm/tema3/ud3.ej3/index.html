Pregunta 1

  • Diferencias entre la etiqueta title y alt

A la hora de trabajar con imagenes emplearemos la etiqueta title para definir el titulo de esta, mientras que la etiqueta alt se emplea para mostrar un texto alternativo en caso de que la visualización no este disponible.

Pregunta 2

  • ¿Para que se utilizan los elementos <map> y <area> ?

El elemento <map> se utiliza para definir una imagen mapeada, la cual es una imagen con areas que son clickeables.

El elemento <area> se utiliza para definir cual es el areas clickeables de la imagen que hemos mapeado.
