ACCESO A ODOO

Una vez creada la base de datos de Odoo volvemos a acceder a localhost:8069 donde encontraremos las distintas funciones con las que permite trabajar Odoo. En mi caso para realizar las pruebas seleccionare Sitio Web pulsando el botón instalar.

Tras acceder a la función de Sitio Web nos llevará a una página en la que seleccionaremos el tema con el que queremos que se cree nuestro Sitio Web.


Acceso Completado, acceda al siguiente paso para conocer la edición del Sitio Web.
